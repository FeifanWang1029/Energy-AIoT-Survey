# Paper comparison tables

Cross-paper tables to support **research-gap discovery**. Fill as the literature review progresses.

- **by_energy_metric.md** — Papers grouped by energy metric (mJ/inference, EDP, carbon, throughput/W, etc.).
- **by_dataset.md** — Papers by dataset / benchmark (MLPerf Tiny, custom IoT, etc.).
- **by_layer.md** — Papers by layer: device / communication / edge / cloud / cross-layer.

See [README § Repository Deliverables](../README.md#repository-deliverables-beyond-paper-lists) for the goal: quickly see which metrics, datasets, and layers are over- or under-studied.
